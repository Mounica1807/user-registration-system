# Django User Registration System

A full-stack user registration system with:

- Backend: Django + Django REST Framework + Simple JWT
- Authentication: Email-based login with JWT tokens
- Frontend: Modern, responsive HTML + CSS + JavaScript
- Features: Register, Login, View Profile, Logout
- Password hashing, server-side validation

## Features
- Secure user registration & login
- JWT token-based authentication
- Clean, professional UI
- Profile page with user details

## Tech Stack
- Django 5.x / 6.x
- Django REST Framework
- djangorestframework-simplejwt
- HTML5, CSS3, JavaScript (vanilla)
